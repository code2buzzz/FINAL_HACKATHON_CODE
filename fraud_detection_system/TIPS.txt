Pro Tip for Hackathon Debugging:
If your graph variable is a compiled LangGraph workspace, you can replace .invoke() with .stream() to watch nodes fire off sequentially. It's incredibly useful for showing live execution progress on a presentation slide or in the terminal:

Python
# To stream state mutations node-by-node instead of a single block call:
for event in graph.stream(initial_state):
    for node_name, updated_state in event.items():
        logger.info(f"📍 State modified via Node: [{node_name}]")
